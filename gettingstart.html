Getting Started

I'm using the express-handlebars package

Express-handlebars is a handy package to use Handlebars as the template renderer with the Express server. We worked with the author and extended it seamlessly. Here's the express-secure-handlebars pacakge requiring only 2 LoCs changes to upgrade.

See Instructions

I'm using the Handlebars runtime

Simply use the secure-handlebars package to perform contextual analysis and add context-aware XSS escaping filters to your templates. Then register the filter implementations as Handlebars helpers for compilation and data binding on client-side.

Demo | See Instructions | Learn More

I want the filters only

The filters observe a just-sufficient escaping principle to escape only those characters sensitive to the specific output context. It's generally faster and more secure than the typical one, that blindly escapes a fixed set of characters regardless of the contexts.

Demo | See Instructions | Learn More
